The category table likes this:
ItemId(int),ItemName(string),ItemNodePath(string),ItemParentId(int),IsLastNode(bool)
Options:
data:json.just get the categories data from database and then encode to json format simply.
ItemId:string.the default value is ItemId.If your table field is not this,you need to define this value.
ItemName:string.the default value is ItemName.If your table field is not this,you need to define this value.
ItemNodePath:string.the default value is ItemNodePath.If your table field is not this,you need to define this value.
ItemParentId:string.the default value is ItemParentId.If your table field is not this,you need to define this value.
IsLastNode:string.the default value is IsLastNode.If your table field is not this,you need to define this value.
INPSeparator:string.default value is , .
DefaultOptionText:string.
DefaultOptionValue:int.
DefaultPID:int.
MaxNode:int.
SelectedId:int.optional.
LastValueId:string.optional.the container id where you want to set the last id .
ShowTargetId:string.optional.the container id where you want to show the selected texts after the select box change.
ShowSeparator:string.
Prefix:string.optional.
callback:object,callable function name.default value is null.
